\# Task 2: Sentiment Analysis using Machine Learning



\## Overview



This task focuses on building a \*\*Sentiment Analysis pipeline\*\* using Machine Learning techniques as part of the Generative AI Module. The objective is to classify text data into sentiment categories by applying preprocessing, feature extraction, and model training.



\## Features



\* Text preprocessing using NLP techniques

\* Feature extraction using \*\*Bag of Words (BoW)\*\* and \*\*TF-IDF\*\*

\* Model building using:



&#x20; \* Logistic Regression

&#x20; \* Decision Tree

\* Performance evaluation using:



&#x20; \* Accuracy

&#x20; \* Precision

&#x20; \* Recall

&#x20; \* F1 Score

\* Comparative analysis of models and vectorization techniques



\## Tech Stack



\* Python

\* Scikit-learn

\* NLTK

\* Pandas \& NumPy



\## Key Highlights



\* Compared multiple models to identify best performance

\* Observed impact of different feature extraction techniques

\* Decision Tree with BoW achieved the best performance among tested models



\## Key Takeaway



Demonstrates how preprocessing and feature engineering directly impact model performance in NLP tasks, and highlights the importance of selecting the right model and representation technique for sentiment classification.
