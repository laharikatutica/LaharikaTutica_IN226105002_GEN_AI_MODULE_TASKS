# Task 3: Chatbot using Transformers

## Objective
To build a conversational chatbot using a pre-trained transformer model from Hugging Face.

## Description
This chatbot takes user input, processes it using a transformer model, and generates meaningful responses. It maintains conversation flow and simulates a basic AI assistant.

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## How to Run
1. Install dependencies:
   pip install transformers torch

2. Open the notebook and run all cells.

## Conclusion
This project demonstrates how transformer models can be used to build simple and effective conversational chatbots.
